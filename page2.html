The original solution:

While proof of work was the basis on which e-currencies were built it is by no means perfect. This makes currencies such as Bitcoin less efficient and very power intensive to mine

If a currency is mined using proof of stake every single miner in the block chain must process a transaction when it happens. This is very inefficient. While this did have apparent security benefits initially and seemed to be the only way to guarantee a secure transaction it is not anymore today.


The future of e-currencies:

Proof of stake is a much better solution to how to mine e-currencies. This allows currencies such as Ethereum to be much more efficient than older currencies such as Bitcoin

A proof of stake currency on the other hand is much more efficient as not every miner in the network must process a transaction when it happens. Instead, the miners essentially bid for the ability to process the transaction. This allows for different miners to procces different new transactions.
